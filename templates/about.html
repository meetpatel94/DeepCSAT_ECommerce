Customer Satisfaction Prediction System

AI-powered system that analyzes customer support data and predicts CSAT scores using an Artificial Neural Network. Transform customer experience with data-driven insights.

Project Objective

The objective of this project is to analyze customer support interactions and predict customer satisfaction scores using machine learning techniques. This helps businesses understand and improve customer experience through actionable insights and real-time predictions.

Dataset Features

The dataset contains customer support interaction records including:

  • Channel name & type
  • Product category
  • Customer city
  • Agent shift timing
  • Item price range
  • CSAT score (1-5)
