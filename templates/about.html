Project Objective

The main objective of this project is to analyze customer support data and predict customer satisfaction (CSAT) scores using an Artificial Neural Network (ANN). This helps organizations understand customer experience and improve service quality.

Dataset Description

The dataset contains customer support interactions from an e-commerce platform, including features such as communication channel, product category, customer city, agent shift, and item price.

Total Records: 85,000+ customer interactions collected from real e-commerce support data

Machine Learning Model

An Artificial Neural Network (ANN) model was trained to predict CSAT scores based on customer interaction features. The model uses multiple input features including channel name, product category, customer city, agent shift, and item price — learning complex non-linear patterns to deliver accurate predictions.

System Features
Customer dataset filtering and exploration with pagination
AI-based CSAT score prediction using trained ANN model
Interactive analytics dashboard with visual charts
Visualization of satisfaction trends across cities and products
Conclusion

This system demonstrates how Artificial Intelligence and Deep Learning can help businesses analyze customer feedback and predict satisfaction levels. The dashboard provides insights that support decision-making and improve overall customer service quality across channels and product categories.
